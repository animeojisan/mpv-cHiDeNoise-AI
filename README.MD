mpv-cHiDeNoise-AI_GitHub
=======================

このフォルダは、mpv-cHiDeNoise-AI 関連の自作ファイルや、
編集済みファイルを GitHub で管理していくための個人用フォルダです。

現時点では、どのファイルが完全な自作で、
どのファイルが既存ファイルの流用・改変にあたるのかを
まだ完全には整理できていません。
そのため、今後少しずつ確認しながら整理を進めていく予定です。

これまでは独学で試行錯誤しながらファイルを追加・編集してきたため、
一部ファイルについては、出所やライセンス面の整理が
十分ではない可能性があります。
今後はその点にも配慮し、自作・改変・流用の区別や、
必要に応じた整理を少しずつ進めていく予定です。

また、これまでの制作では多くの既存ツールやアプリ、
オープンソースの成果物を参考にしてきました。
それらを公開・共有してくださった制作者の方々に、
敬意と感謝を表します。
今後はそうした先行する成果へのリスペクトを大切にしながら、
必要な整理も進めていきたいと考えています。

ライセンス・参考元・出所整理については NOTICE.txt を参照してください。
mpv_PlayKit 由来ファイルに関する上流ライセンス情報は UPSTREAM_LICENSES.txt を参照してください。
Python 系第三者コンポーネントに関するライセンス情報は THIRD_PARTY_LICENSES_python.txt を参照してください。


【主な管理対象】
・.cpp
・Lua
・input.conf
・mpv.conf
・.vpy / .py
・各種メモ、説明書

【主な管理対象外】
・exe
・ログ
・cache
・engine
・大容量モデル
・一時生成物


まずは「今後も残したいもの」「自分で触ったもの」を
失わないことを目的として管理を始めています。
本リポジトリは、mpv-cHiDeNoise-AI 関連のソース、設定、スクリプト類の管理を目的としています。
配布版アプリ全体とは構成が異なるため、クローンのみでそのまま完成版と同じ状態にはなりません。
完成版の配布は Releases および外部リンクを参照してください。
