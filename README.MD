mpv-cHiDeNoise-AI_GitHub

mpvベースのAI再生プレーヤー「mpv-cHiDeNoise-AI」関連の
自作ファイルや編集済みファイルを GitHub で管理していくための個人用リポジトリです。

mpv-cHiDeNoise-AI の紹介動画:
https://www.youtube.com/watch?v=xrgJoh29bpM

このリポジトリでは、主に以下のようなファイルを管理しています。

- 自作ファイル
- 編集・改変したファイル
- 設定ファイル
- スクリプト類
- メモや説明書
- 現在整理中の関連ファイル

主な管理対象:
- .cpp
- Lua
- input.conf
- mpv.conf
- .vpy / .py
- 各種メモ、説明書
- 自作または編集済みの補助ツール関連ファイル

主な管理対象外:
- exe
- ログ
- cache
- engine
- 大容量モデル
- 一時生成物
- 完成版配布用 zip 一式

このリポジトリには、自作ファイル、既存プロジェクトをベースに改変したファイル、
現在出所やライセンスを整理中のファイルが混在しています。

これまでは独学で試行錯誤しながらファイルを追加・編集してきたため、
一部ファイルについては、出所やライセンス面の整理が十分ではない可能性があります。
今後はその点にも配慮し、自作・改変・流用の区別や必要な整理を少しずつ進めていく予定です。

多くの既存ツール、アプリ、オープンソースの成果物、
先行する制作者の知見や工夫を参考にしています。
それらを公開・共有してくださった制作者・貢献者の方々に敬意と感謝を表します。

主な参考元・ベースの一つ:
- hooke007/mpv_PlayKit
  https://github.com/hooke007/mpv_PlayKit

ライセンス、参考元、出所整理に関する補足:
- NOTICE.txt
- UPSTREAM_LICENSES.txt
- THIRD_PARTY_LICENSES_python.txt

このリポジトリは mpv-cHiDeNoise-AI 関連ファイルの管理用であり、
クローンしただけで配布版アプリ全体がそのまま再現される構成ではありません。
完成版の配布については Releases および外部配布リンクを参照してください。

input.conf は日本語版を基準として管理しています。
必要に応じて、英語版や派生版を別途用意する場合があります。

本アプリおよび関連ファイルは個人制作・個人配布のため、
ご利用は各自の判断と責任でお願いいたします。
利用環境や設定内容の違いにより、正常に動作しない場合や、
意図しない不具合が発生する可能性があります。

また、本リポジトリは個人のペースで管理・更新しています。
一般的な要望、サポート、ディスカッションには対応していません。

一部の過去作成ファイルについては、
現時点で詳細な処理内容の整理が未完了です。
今後少しずつ確認・整理を進めていく予定です。

まずは「今後も残したいもの」「自分で触ったもの」を
失わないことを目的として管理を始めています。
