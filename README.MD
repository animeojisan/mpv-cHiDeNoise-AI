# \## このリポジトリについて

# 

# このリポジトリは、mpv-cHiDeNoise-AI 関連の以下のようなファイルを  

# 整理・管理していくことを目的としています。

# 

# \- 自作ファイル

# \- 自分で編集・改変したファイル

# \- 今後も残しておきたい設定ファイルや補助ファイル

# \- 現在整理中のファイル

# 

# 現時点では、どのファイルが完全な自作で、  

# どのファイルが既存ファイルの流用・改変にあたるのかを  

# まだ完全には整理できていません。  

# そのため、今後少しずつ確認しながら整理を進めていく予定です。

# 

# これまでは独学で試行錯誤しながらファイルを追加・編集してきたため、  

# 一部のファイルについては、出所やライセンス面の整理が  

# 十分ではない可能性があります。  

# 今後はその点にも配慮し、自作・改変・流用の区別や  

# 必要な整理を少しずつ進めていく予定です。

#
